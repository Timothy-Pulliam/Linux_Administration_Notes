== Bash Completion ==
Install Bash Completion does not come preinstalled on CentOS/RHEL minumum systems. You must install it.
$ yum install bash-completion

==== Setting the BASH Prompt (PS1, PS2, PS3) ====
1) man bash
2) search "PS1"
3) search "PROMPTING"
